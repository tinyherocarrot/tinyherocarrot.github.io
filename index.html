Chews

Chews is created to personalize travelers' dining experience and humanize local cultures, by allowing travelers to connect and schedule meals with local foodies. Users are able to securely login with Auth0, create and book "bites", and message each other in-app.

CONTRIBUTION

I helped to develop the front end on Chews, from outlining user stories to wireframing to implementation using React and styled-components. This was my first large-scale project using React, and I soon grew to love its highly modular approach to building user interface.

Qlink
